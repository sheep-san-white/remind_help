# remind_help

## このリポジトリについて

`Slack`の`/remind`コマンドのオプションが分かりづらいのでヘルプを参照するための`/remind_help`コマンドを作りました。

## 必要な権限

* `G Suite`または`Gmail`のアカウントを持っていて`Google Apps Script`を操作する権限がある人
